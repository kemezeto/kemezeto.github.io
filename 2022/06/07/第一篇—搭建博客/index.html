第一篇——搭建博客

第一章（准备工作）

1.安装git

  • 官网下载: git
  • 下载较慢请用镜像网站
  • 安装后验证 git -v
  • 注：验证方法为右键选择 git bush here 然后输入

    2.安装node.js

  • 官网下载: node.js
  • 安装后验证 node -v

    3.命令行安装cnpm

  • 输入：npm install -g cnpm --registry==https://registry.npm.taobao.org
  • 安装后验证 cnpm -v

    4.命令行安装hexo

  • 输入：cnpm install -g hexo-cli
  • 安装后验证 hexo -v

第二章（配置github）

1.在github上创建仓库

1.创建

  • 首先你需要注册一个github账号，并且进行了邮箱验证。
  • 点击右上角+，然后点击New repository，新建一个名为你的用户名.github.io的仓库，例：https://用户名.github.io/

2.配置SSH免密登录

  • 找到位于电脑C:\Users\用户名\下的（.ssh）文件夹并删除
  • 在上行地址中右键打开Git Bash Here输入命令 ssh-keygen -t rsa -C 你的github邮箱地址 ，生成ssh密钥，输入后连敲三次回车。
  • 打开新生成的.ssh文件夹，打开id_rsa.pub文件，复制里面的内容。
  • 打开github主页，然后个人设置->SSH and GPG keys -> New SSH key，点击粘贴，title可输入”个人博客“。
  • 回到文件夹，右键Git Bash Here 输入 ssh -T git@github.com 显示：Are you sure you want to continue connecting (yes/no)? ，输入 yes ,显示Hi liuxianan! You’ve successfully authenticated, but GitHub does not provide shell access ,配置成功。
  • 右键Git Bash Here，输入 git config --global user.name "你的github用户名" ，然后输入 git config --global 你的github验证邮箱 。

第三章（使用hexo搭建博客）

1.初始化

  • 新建名为hexo的文件夹，牢记这个文件夹的位置，很重要
  • hexo文件夹下右键Git Bash Here，输入 hexo init 初始化
  • 输入 hexo g 和 hexo s ,初始化完成

    2.将博客部署到github个人主页上

  • hexo文件夹右键Git Bash Here，输入 npm install hexo-deployer-git --save
  • 打开_config.yml文件，在文件末尾下添加如下内容：
    deploy:
    type: git
    repository: git@github.com:github用户名/github用户名.github.io.git
    branch: master
  • 右键Git Bash Here，输入 hexo d 将本地blog推送到githu仓库
  • 完成，浏览器输入https://github用户名.github.io/前往你的博客吧

各种bug解决方案

1.本地部署与远端出现不同

  • 打开_config.yml文件，找到URL
    改为：
    url: http://kemeteto.github.io
    root: /你的hexo文件夹
